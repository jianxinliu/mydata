# mydata

## 简介

mydata 是一个插件式的通用数据处理工具。可用于对各种类型的数据进行分析、可视化。

## 功能一览

- [ ] 数据源：csv, excel
- [ ] 数据分组
- [ ] 数据过滤
- [ ] 数据操纵
- [ ] 透视图
- [ ] 可视化
- [ ] dashboard 大屏，可以同时展示多种图形
- [ ] 模板程序共享

## 技术选型

- wails：客户端框架
- element-plus,vue3,ts：前端套件
- alasql.js：前端 sql 解析器、执行器
- golang：后端语言
- echarts/go-echarts: 可视化支持

## 架构设计

- 如何模块化
- 计算、存储与逻辑分离
- sql 视图实现串联查询
- 临时表存本地，格式待研究
